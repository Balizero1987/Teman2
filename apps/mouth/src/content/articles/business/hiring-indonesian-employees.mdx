### Cost Calculator

<Calculator
  title="Total Employment Cost"
  description="Estimate employer total cost"
  fields={[
    {
      id: "gross",
      label: "Gross Monthly Salary (IDR)",
      type: "number",
      defaultValue: 15000000,
      min: 0,
      step: 1000000,
      suffix: "IDR"
    },
    {
      id: "thr",
      label: "Include THR Provision",
      type: "select",
      options: [
        { value: "yes", label: "Yes" },
        { value: "no", label: "No" }
      ],
      defaultValue: "yes"
    }
  ]}
  calculateResult={(values) => {
    const gross = Number(values.gross);

    // BPJS employer contributions
    const bpjsKesehatan = gross * 0.04; // 4% health
    const bpjsJKK = gross * 0.0024; // 0.24% min work accident
    const bpjsJKM = gross * 0.003; // 0.3% death
    const bpjsJHT = gross * 0.037; // 3.7% pension fund
    const bpjsJP = gross * 0.02; // 2% pension

    const totalBPJS = bpjsKesehatan + bpjsJKK + bpjsJKM + bpjsJHT + bpjsJP;

    // THR provision (1 month spread over 12)
    const thrProvision = values.thr === "yes" ? gross / 12 : 0;

    const totalMonthly = gross + totalBPJS + thrProvision;

    return {
      total: Math.round(totalMonthly),
      breakdown: [
        { label: "Gross Salary", amount: gross },
        { label: "BPJS Kesehatan (4%)", amount: Math.round(bpjsKesehatan) },
        { label: "BPJS JKK (0.24%)", amount: Math.round(bpjsJKK) },
        { label: "BPJS JKM (0.3%)", amount: Math.round(bpjsJKM) },
        { label: "BPJS JHT (3.7%)", amount: Math.round(bpjsJHT) },
        { label: "BPJS JP (2%)", amount: Math.round(bpjsJP) },
        { label: "THR Provision", amount: Math.round(thrProvision) },
        { label: "Total Monthly Cost", amount: Math.round(totalMonthly) }
      ],
      currency: "IDR",
      note: `Employer BPJS: ~10.24% of salary`
    };
  }}

## BPJS Registration

### BPJS Kesehatan (Health)

<InfoCard variant="note" title="Health Insurance">
**Contributions:**
- Total: 5% of salary
- Employer: 4%
- Employee: 1%

**Cap:** IDR 12 million base salary

**Coverage:**
- Employee
- Spouse
- Up to 3 children
- Additional family members: +1% each
</InfoCard>

### BPJS Ketenagakerjaan (Employment)

| Program | Employer | Employee | Total |
|---------|----------|----------|-------|
| JKK (Work Accident) | 0.24-1.74% | 0% | Varies |
| JKM (Death) | 0.30% | 0% | 0.30% |
| JHT (Pension Fund) | 3.70% | 2% | 5.70% |
| JP (Pension) | 2% | 1% | 3% |

### Contract Checklist

- Employee and employer details
- Job title and description
- Workplace location
- Working hours
- Salary and components
- Benefits
- Start date (and end date for PKWT)
- Probation terms (PKWTT only)
- Termination provisions
- Signed by both parties

## Probation Period

<InfoCard variant="warning" title="Probation Rules">
**For PKWTT (Permanent) only:**
- Maximum 3 months
- Must be in writing
- Full salary during probation
- Can terminate with notice during probation
- No severance if terminated during probation

**For PKWT (Fixed-term):**
- Probation NOT allowed
- Any probation clause is void
</InfoCard>

## Payroll Considerations

### Payment Requirements

| Requirement | Detail |
|-------------|--------|
| Frequency | Monthly (most common) |
| Method | Bank transfer |
| Currency | IDR |
| Pay slip | Required |
| Deductions | BPJS, PPh 21, loans |

### Tax Withholding (PPh 21)

Employer must:
1. Calculate monthly PPh 21
2. Withhold from salary
3. Remit by 10th of following month
4. Report by 20th of following month
5. Issue annual tax slip (1721-A1)

## Common Mistakes

| Mistake | Impact |
|---------|--------|
| No written contract | Defaults to PKWTT |
| Late BPJS registration | Fines and penalties |
| Using PKWT for permanent roles | Contract becomes PKWTT |
| Probation in PKWT | Clause is void |
| Not collecting NPWP | Tax reporting issues |
| Missing tax withholding | Employer liability |
