## PKWT (Fixed-Term Contract)

<InfoCard variant="warning" title="PKWT Limitations">
**PKWT can only be used for:**
- Temporary work
- Seasonal work
- One-time projects
- Work related to new products/processes (max 2 years)

**Cannot be used for:**
- Permanent positions
- Core ongoing operations
- Indefinite assignments

Violation: Contract automatically becomes PKWTT (permanent).
</InfoCard>

### PKWT Duration Rules

| Rule | Limit |
|------|-------|
| Initial term | Max 5 years (cumulative) |
| Extensions | Within 5-year limit |
| No probation | Probation void if written |
| Compensation | Required upon completion |

### Overtime Rules

<Calculator
  title="Overtime Calculator"
  description="Calculate overtime pay"
  fields={[
    {
      id: "monthly",
      label: "Monthly Salary (IDR)",
      type: "number",
      defaultValue: 10000000,
      min: 0,
      step: 500000,
      suffix: "IDR"
    },
    {
      id: "hours",
      label: "Overtime Hours",
      type: "number",
      defaultValue: 10,
      min: 0,
      max: 14,
      suffix: "hours"
    },
    {
      id: "day",
      label: "Day Type",
      type: "select",
      options: [
        { value: "weekday", label: "Regular weekday" },
        { value: "weekend", label: "Weekend/holiday" }
      ],
      defaultValue: "weekday"
    }
  ]}
  calculateResult={(values) => {
    const monthly = Number(values.monthly);
    const hours = Number(values.hours);

    // Hourly rate = 1/173 of monthly salary
    const hourlyRate = monthly / 173;

    let overtime = 0;

    if (values.day === "weekday") {
      // First hour: 1.5x, subsequent hours: 2x
      if (hours >= 1) {
        overtime += hourlyRate * 1.5;
        if (hours > 1) {
          overtime += (hours - 1) * hourlyRate * 2;
        }
      }
    } else {
      // Weekend/holiday: different calculation
      // First 8 hours: 2x, 9th hour: 3x, 10th+: 4x
      const first8 = Math.min(hours, 8);
      overtime += first8 * hourlyRate * 2;
      if (hours > 8) {
        overtime += hourlyRate * 3;
        if (hours > 9) {
          overtime += (hours - 9) * hourlyRate * 4;
        }
      }
    }

    return {
      total: Math.round(overtime),
      breakdown: [
        { label: "Monthly Salary", amount: monthly },
        { label: "Hourly Rate (1/173)", amount: Math.round(hourlyRate) },
        { label: "Overtime Hours", amount: hours },
        { label: "Overtime Pay", amount: Math.round(overtime) }
      ],
      currency: "IDR",
      note: values.day === "weekday" ? "Weekday: 1.5x first hour, 2x after" : "Weekend: 2x first 8, 3x 9th, 4x 10th+"
    };
  }}
/>

## Minimum Wage

### Regional Minimum Wage (UMR/UMK)

| Province/City | 2026 UMR (Approx) |
|---------------|-------------------|
| DKI Jakarta | ~IDR 5.1M |
| Bali | ~IDR 2.9M |
| Surabaya | ~IDR 4.7M |
| Bandung | ~IDR 4.0M |
| Semarang | ~IDR 3.1M |

<InfoCard variant="note" title="Minimum Wage Notes">
- UMR is minimum for permanent employees
- Updated annually (usually January)
- Sectoral minimums may apply
- Components: basic salary can be 75%, allowances 25%
- PKWT workers entitled to same minimums
</InfoCard>

## Company Regulations

### When Required

<InfoCard variant="note" title="Peraturan Perusahaan">
Companies with **10+ employees** must have:
- **PP (Peraturan Perusahaan)** - Company regulations

Or with union:
- **PKB (Perjanjian Kerja Bersama)** - Collective bargaining agreement

Must be registered with Manpower office and renewed every 2 years.
</InfoCard>
