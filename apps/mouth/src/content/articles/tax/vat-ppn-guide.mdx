# VAT (PPN) in Indonesia 2026: Complete Business Guide

Pajak Pertambahan Nilai (PPN) is Indonesia's Value Added Tax. Understanding it is crucial for business compliance.

<InfoCard variant="warning" title="Aggiornamento VAT 2026 (Fonte: pajak.go.id)">
**PPN 12% in vigore dal 1 gennaio 2025:**

L'IVA indonesiana è ufficialmente al **12%** dal 1 gennaio 2025. Tuttavia, grazie al meccanismo **DPP Nilai Lain** (Base Imponibile Altra), l'aliquota effettiva per la maggior parte dei beni e servizi rimane all'**11%**.

- **Beni normali**: 12% × (11/12) = **11% effettivo**
- **Beni di lusso**: **12% pieno** (senza DPP Nilai Lain)
- **Tutte le fatture (Faktur Pajak)** devono essere emesse tramite **Coretax/e-Faktur**

**Nota Importante**: Questa struttura è stata implementata per minimizzare l'impatto sui consumatori mantenendo formalmente l'aliquota del 12% prevista dalla legge.
</InfoCard>

## VAT Registration

### Who Must Register?

<InfoCard variant="note" title="PKP Registration Threshold">
You must register as **PKP (Pengusaha Kena Pajak)** if:

- Gross turnover exceeds **IDR 4.8 billion/year**
- Or you **voluntarily register**

As PKP, you must:
- Charge VAT on sales
- Issue tax invoices (Faktur Pajak)
- File monthly VAT returns
- Can claim input VAT credits
</InfoCard>

## VAT Calculation

<Calculator
  title="VAT Calculator"
  description="Calculate VAT on transactions"
  fields={[
    {
      id: "amount",
      label: "Transaction Amount (IDR)",
      type: "number",
      defaultValue: 100000000,
      min: 0,
      step: 1000000,
      suffix: "IDR"
    },
    {
      id: "type",
      label: "Transaction Type",
      type: "select",
      options: [
        { value: "exclusive", label: "Add VAT to price" },
        { value: "inclusive", label: "VAT included in price" }
      ],
      defaultValue: "exclusive"
    }
  ]}
  calculateResult={(values) => {
    const amount = Number(values.amount);
    const rate = 0.11; // 11%

    let baseAmount, vatAmount, totalAmount;

    if (values.type === "exclusive") {
      baseAmount = amount;
      vatAmount = amount * rate;
      totalAmount = amount + vatAmount;
    } else {
      totalAmount = amount;
      baseAmount = amount / (1 + rate);
      vatAmount = totalAmount - baseAmount;
    }

    return {
      total: vatAmount,
      breakdown: [
        { label: "Base Amount (DPP)", amount: Math.round(baseAmount) },
        { label: "VAT (11%)", amount: Math.round(vatAmount) },
        { label: "Total with VAT", amount: Math.round(totalAmount) }
      ],
      currency: "IDR",
      note: "Rate: 11% (standard)"
    };
  }}
/>

## Tax Invoices (Faktur Pajak)

### E-Faktur System

<InfoCard variant="warning" title="E-Faktur Required">
All VAT-registered businesses must use the **e-Faktur** electronic invoicing system:

- Download e-Faktur application
- Register digital certificate
- Issue all invoices electronically
- Upload to DJP system
- Input VAT credits electronically
</InfoCard>

### Faktur Pajak Contents

| Required Field | Description |
|----------------|-------------|
| Seller NPWP | Your tax number |
| Buyer NPWP | Customer tax number |
| Invoice number | Sequential, system-generated |
| Transaction date | Date of supply |
| Description | Goods/services detail |
| DPP | Tax base (price before VAT) |
| PPN | VAT amount |
| Digital signature | Electronic verification |

## Input vs Output VAT

### How VAT Credits Work

| Transaction | VAT Type | Treatment |
|-------------|----------|-----------|
| Sales to customers | Output VAT | Collect and remit |
| Purchases from suppliers | Input VAT | Claim as credit |
| Net position | Difference | Pay if output > input |
| Net credit | Difference | Carry forward or refund |

### Example Monthly Calculation

| Description | Amount |
|-------------|--------|
| Output VAT (sales) | IDR 110M |
| Input VAT (purchases) | IDR 55M |
| Net VAT payable | IDR 55M |

Some goods and services are VAT exempt:

## Common Issues

### VAT Problems to Avoid

| Issue | Consequence |
|-------|-------------|
| Late filing | IDR 500K penalty |
| Incorrect invoices | Credits disallowed |
| Missing invoices | Unable to claim credits |
| Wrong NPWP | Invoice rejected |
| Late payment | Interest penalties |

### Input VAT Credit Rules

<InfoCard variant="warning" title="Claimable Input VAT">
Not all input VAT is creditable. Credits are **denied** for:

- Entertainment expenses
- Non-business purchases
- Purchases without valid tax invoice
- Purchases from non-PKP suppliers
- Luxury goods (cars, etc.)
</InfoCard>

## VAT Refund

### When Refund Available

- Output VAT consistently less than input VAT
- Export businesses (zero-rated sales)
- New investments with large input VAT

### Refund Process

1. File annual or special refund request
2. Tax audit typically required
3. Processing time: 1-12 months
4. Full documentation required
