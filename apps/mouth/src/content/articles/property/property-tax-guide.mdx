## Annual Property Tax (PBB)

### What is PBB?

<InfoCard variant="note" title="Pajak Bumi dan Bangunan">
**PBB** (Land and Building Tax) is:
- Annual tax on property ownership
- Based on government-assessed value (NJOP)
- Paid by property owner
- Due by August each year

All landowners and building owners must pay PBB.
</InfoCard>

### PBB Calculation

| Component | Explanation |
|-----------|-------------|
| NJOP | Government assessed value |
| NJOPTKP | Tax-free threshold (varies by region) |
| Taxable value | NJOP minus NJOPTKP |
| Tax rate | 0.1-0.5% (varies by region and value) |

### PBB Calculator

<Calculator
  title="PBB Estimate"
  description="Estimate your annual property tax"
  fields={[
    {
      id: "njop",
      label: "NJOP Value (IDR)",
      type: "number",
      defaultValue: 500000000,
      min: 0
    },
    {
      id: "region",
      label: "Region",
      type: "select",
      options: [
        { value: "jakarta", label: "Jakarta" },
        { value: "bali", label: "Bali" },
        { value: "other", label: "Other" }
      ],
      defaultValue: "bali"
    }
  ]}
  calculateResult={(values) => {
    // NJOPTKP varies by region
    const njoptkp = {
      jakarta: 60000000,
      bali: 50000000,
      other: 30000000
    };

    const threshold = njoptkp[values.region];
    const taxable = Math.max(0, values.njop - threshold);

    // Simplified rate (actual varies by bracket)
    let rate;
    if (values.njop < 1000000000) {
      rate = 0.001; // 0.1%
    } else if (values.njop < 2000000000) {
      rate = 0.002; // 0.2%
    } else {
      rate = 0.003; // 0.3%
    }

    const tax = taxable * rate;

    return {
      total: Math.round(tax),
      breakdown: [
        { label: "NJOP Value", amount: values.njop },
        { label: "NJOPTKP (threshold)", amount: threshold },
        { label: "Taxable value", amount: taxable },
        { label: "Effective rate", amount: `${(rate * 100).toFixed(2)}%` },
        { label: "Annual PBB", amount: Math.round(tax) }
      ],
      currency: "IDR",
      note: "Actual tax may vary based on local regulations"
    };
  }}
/>

## Transfer/Acquisition Tax (BPHTB)

### What Buyers Pay

<InfoCard variant="warning" title="BPHTB - 5% Transfer Tax">
**BPHTB** applies when you acquire property:
- Purchase
- Gift
- Inheritance
- Exchange

**Rate:** 5% of transfer value minus NJOPTKP
**Paid by:** Buyer/recipient
**When:** Before title transfer
</InfoCard>

### BPHTB Calculation

| Element | Calculation |
|---------|-------------|
| Transaction value | Higher of price or NJOP |
| Minus NJOPTKP | Tax-free threshold |
| Tax base | Result |
| BPHTB | 5% of tax base |

### Example

| Item | Amount |
|------|--------|
| Purchase price | IDR 2,000,000,000 |
| NJOP | IDR 1,500,000,000 |
| Use higher value | IDR 2,000,000,000 |
| NJOPTKP | IDR 60,000,000 |
| Tax base | IDR 1,940,000,000 |
| **BPHTB (5%)** | **IDR 97,000,000** |

## Seller's Tax (PPh Final)

### What Sellers Pay

| Transaction | Tax Rate |
|-------------|----------|
| Property sale | 2.5% of transaction value |
| Land sale | 2.5% of transaction value |

<InfoCard variant="note" title="Seller Tax">
**PPh Final 2.5%** is:
- Paid by seller
- Based on transaction value (or NJOP if higher)
- Due before title transfer
- Final (no further income tax on gain)
</InfoCard>

## Rental Income Tax

### Tax on Rent Received

<InfoCard variant="note" title="10% Final Tax">
Rental income from property is subject to:

**10% Final PPh** on gross rental income

- Applies to land and buildings
- No deductions for expenses
- Final (not added to other income)
- Should be withheld by corporate tenants
</InfoCard>

### Who Withholds?

| Tenant Type | Tax Treatment |
|-------------|---------------|
| Corporate tenant | Must withhold 10% |
| Individual tenant | Landlord self-assesses |

### Reporting

| Situation | Action |
|-----------|--------|
| Corporate tenant | They issue bukti potong |
| Individual tenant | You report and pay |
| Mixed | Declare all in annual SPT |

## NJOP Explained

### What is NJOP?

<InfoCard variant="note" title="Government Assessed Value">
**NJOP** (Nilai Jual Objek Pajak) is:
- Government-assessed property value
- Used for tax calculations
- Usually lower than market value
- Updated periodically

Find your NJOP on your SPPT (tax notice) or check at local tax office.
</InfoCard>

### NJOP vs Market Value

| Aspect | NJOP | Market Value |
|--------|------|--------------|
| Set by | Government | Market |
| Typical level | 50-80% of market | 100% |
| Update frequency | Periodic | Real-time |
| Used for | Taxes | Transactions |

## Tax Planning Tips

### Legal Ways to Minimize

| Strategy | How It Works |
|----------|--------------|
| Time acquisition | Year-end may allow planning |
| Structure ownership | PT PMA vs personal |
| Use NJOPTKP | Ensure threshold applied |
| Clear PBB on time | Avoid penalties |
| Document expenses | For future sale basis |

<InfoCard variant="warning" title="Tax Compliance">
**Don't:**
- Under-report transaction values
- Hide property ownership
- Fail to pay PBB
- Ignore rental income

Tax authorities cross-reference with BPN and notary records. Non-compliance creates problems when you want to sell.
</InfoCard>
