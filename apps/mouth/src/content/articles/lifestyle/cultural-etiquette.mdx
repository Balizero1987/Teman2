# Indonesian Cultural Etiquette 2026: Expat Guide

Indonesia has rich cultural traditions. Understanding local customs will help you build better relationships and avoid misunderstandings.

## Core Cultural Values

<InfoCard variant="note" title="Key Values">
**Indonesian society values:**

- **Harmony** - Avoiding conflict, maintaining peace
- **Respect** - For elders, hierarchy, religion
- **Community** - Group over individual
- **Face** - Preserving dignity, avoiding embarrassment
- **Patience** - "Rubber time" is real
- **Hospitality** - Welcoming to guests

Understanding these values explains many customs.
</InfoCard>

---

## Social Etiquette

### Greetings

| Situation | Appropriate Greeting |
|-----------|---------------------|
| Formal/Business | Slight bow, handshake |
| Casual | Smile, wave, "Halo" |
| Elders | Slight bow, respectful tone |
| Religious Muslims | Same-gender handshake preferred |
| Balinese | "Om Swastiastu" (greeting) |

### Physical Contact

| Behavior | Acceptability |
|----------|---------------|
| Handshake | Normal (same gender) |
| Pat on back | Between friends |
| Touching head | Never (sacred) |
| Pointing with finger | Avoid (use thumb instead) |
| Left hand for giving | Avoid (use right hand) |
| Public affection | Keep minimal |

<InfoCard variant="warning" title="The Head">
**The head is sacred** in Indonesian culture.

- Never touch someone's head
- Includes children and friends
- Even playful pats are disrespectful
- Hair styling services are exception (professional context)
</InfoCard>

|------|-----|

| Storing | Place carefully, not in back pocket |

### Business Relationships

<InfoCard variant="tip" title="Relationship First">
**In Indonesia:**

- Business often follows relationship
- Small talk before business matters
- Lunch/dinner meetings important
- Building trust takes time
- Personal relationships matter

Don't rush to business - invest in relationships first.

## Religious Sensitivity

### Islam (Majority Religion)

| Consideration | Guidance |
|---------------|----------|
| Friday prayers | Meetings may pause |
| Ramadan | Avoid eating in public |
| Pork/alcohol | Don't offer to Muslims |
| Right hand | Use for giving/receiving |
| Dress code | Conservative appreciated |
| Mosques | Remove shoes, cover up |

### Hinduism (Bali)

| Consideration | Guidance |
|---------------|----------|
| Temples | Proper dress required |
| Offerings | Don't step over |
| Ceremonies | Show respect |
| Menstruation | Temple entry restricted |
| Holy days | Some closures |

<InfoCard variant="warning" title="Temple Dress Code">
**Visiting Balinese temples:**

- Sarong required (often provided)
- No shorts/short skirts
- Shoulders covered
- Sash around waist
- Remove shoes often

Many temples have rentals available.
</InfoCard>

## Dining Etiquette

### Table Manners

| Practice | Standard |
|----------|----------|
| Wait for elders | Before eating |
| Right hand | For eating (if using hands) |
| Refuse at first | Common to politely decline |
| Finish food | Shows appreciation |
| Host pays | Usually the inviter |
| Bill splitting | Not common |

### Food Customs

| Situation | Guidance |
|-----------|----------|
| Offered food | Try a bit, even if full |
| Dietary needs | Explain politely |
| Street food | Enjoy! Very common |
| Tipping | Not mandatory, appreciated |
| Alcohol | With Muslims, be discreet |

## Common Dos and Don'ts

### Dos

| Do | Why |
|----|-----|
| Smile often | Opens doors |
| Use titles | Shows respect |
| Accept hospitality | Refusing can offend |
| Learn Bahasa basics | Much appreciated |
| Be patient | Rushing is rude |
| Dress modestly | Respect local norms |
| Remove shoes | When entering homes |

### Don'ts

| Don't | Why |
|-------|-----|
| Lose temper | Causes loss of face |
| Point with finger | Use thumb instead |
| Show soles of feet | Considered disrespectful |
| Touch heads | Sacred body part |
| Rush relationships | Trust takes time |
| Criticize publicly | Causes embarrassment |
| Be too loud | Discretion valued |
