<Calculator
  title="Transfer Cost Comparison"
  description="Compare costs for your transfer amount"
  fields={[
    {
      id: "amount",

      type: "number",

      ],
      defaultValue: "wise"

    // Fee structures (approximate)
    const fees = {
      "wise": { fixed: 3, percent: 0.006, spread: 0 },
      "bank": { fixed: 35, percent: 0, spread: 0.03 },
      "western": { fixed: 5, percent: 0.035, spread: 0.02 },
      "paypal": { fixed: 0, percent: 0.04, spread: 0.025 }
    };

    const fee = fees[values.method];
    const transferFee = fee.fixed + (amount * fee.percent);
    const spreadCost = amount * fee.spread;
    const totalCost = transferFee + spreadCost;
    const received = amount - totalCost;

    // Approximate IDR conversion (mid-market ~16,000)
    const rate = 16000 * (1 - fee.spread);
    const idrReceived = received * rate;

    return {
      total: totalCost,
      breakdown: [
        { label: "Transfer Fee", amount: transferFee },
        { label: "Exchange Rate Spread", amount: spreadCost },

        { label: "You Receive", amount: received }
      ],
      currency: "USD",
      note: `Approximately IDR ${Math.round(idrReceived).toLocaleString()} received`
    };
  }}

## Wise (TransferWise)

<InfoCard variant="tip" title="Best Overall Choice">
**Why Wise is popular:**
- Uses real mid-market exchange rate
- Low, transparent fees
- Fast (usually 1-2 days)
- Easy to use app
- Can receive to Indonesian bank account
</InfoCard>

### How to Use Wise for Indonesia

1. Create Wise account (free)
2. Verify identity
3. Add recipient's Indonesian bank details
4. Send from your bank/card
5. Recipient gets IDR in their account

### Wise Fees

| Amount | Approximate Fee | You Receive |
|--------|-----------------|-------------|
| $500 | ~$5 | ~IDR 7.9M |
| $1,000 | ~$8 | ~IDR 15.9M |
| $2,000 | ~$14 | ~IDR 31.8M |
| $5,000 | ~$30 | ~IDR 79.5M |

## Bank Wire Transfers

### SWIFT Transfers

Traditional bank wires still work but are expensive:

| Bank | Fee | Notes |
|------|-----|-------|
| Chase | $40-50 | International wire |
| Bank of America | $35-45 | Varies by account |
| Wells Fargo | $35-45 | Standard rate |
| HSBC | $30-40 | Better for HSBC to HSBC |

### Hidden Costs

<InfoCard variant="warning" title="Watch for Hidden Fees">
Bank transfers have multiple costs:
- **Sending bank fee**: $25-50
- **Receiving bank fee**: $5-20
- **Correspondent bank fees**: $10-30
- **Exchange rate markup**: 2-4%

A $2,000 transfer might cost $100+ total!
</InfoCard>

## Digital Payment Alternatives

### For Daily Expenses

Instead of transferring everything, consider:

1. **Wise Multi-Currency Card**
   - Hold multiple currencies
   - Spend in IDR
   - ATM withdrawals

2. **Revolut**
   - Similar to Wise
   - Free ATM limits

3. **International Credit Cards**
   - No foreign transaction fee cards
   - Good exchange rates

### ATM Withdrawals

| Card Type | Fee | Rate |
|-----------|-----|------|
| Wise | ~$1 | Mid-market |
| Revolut | Free (limits) | Mid-market |
| Charles Schwab | Rebated | Bank rate |
| Regular cards | $3-5 | +3% markup |

## Tips for Best Rates

1. **Avoid airport exchanges** - Worst rates
2. **Use mid-market rate services** - Wise, Revolut
3. **Transfer larger amounts less often** - Reduce fixed fees
4. **Compare before sending** - Rates change daily
5. **Set up rate alerts** - Transfer when rates are good
6. **Avoid weekends** - Rates frozen, delays possible
