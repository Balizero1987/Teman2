## What to Do in Emergency

### Medical Emergency

1. Call 118 or hospital directly
2. State location clearly
3. Describe emergency
4. Stay on line
5. Consider faster transport if appropriate

### Crime/Safety

1. Call 110 (police)
2. Get to safe location
3. Contact embassy if serious
4. File police report
5. Inform insurance

### Lost Passport

1. File police report
2. Contact embassy
3. Get emergency travel document
4. Immigration notification if overstaying
